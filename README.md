# ProfProfCards
Projeto para gestão de cartões de visitas

Este projeto foi desenvolvido como avaliação parcial do bootcamp Banco Inter promovido pela Digital Innovation One. Foram utilizadas, dentre outras, as seguintes tecnologias:
- RecyclerView
- Room database
- Fragments
- NavigationView
- Sqlite

Foram criadas as funções completas de um CRUD e também a funcionalidade de compartilhamento de cartões. Alguns aspectos visuais também foram implementados para melhorar a experiência do usuário.
